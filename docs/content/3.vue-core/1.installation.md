Install the shared package and Convex, then register the plugin during app startup.

::code-group{sync="pm"}

```bash [pnpm]
pnpm add @onmax/convex-vue convex
```

```bash [npm]
npm install @onmax/convex-vue convex
```

```bash [yarn]
yarn add @onmax/convex-vue convex
```

::

## Register the plugin

```ts [src/main.ts]
import { convexVue } from '@onmax/convex-vue'
import { createApp } from 'vue'
import App from './App.vue'

const app = createApp(App)

app.use(convexVue, {
  url: import.meta.env.VITE_CONVEX_URL,
  server: true,
})

app.mount('#app')
```

## Plugin options

```ts
interface ConvexVueOptions {
  url?: string
  clientOptions?: ConvexClientOptions
  server?: boolean
}
```

The plugin defaults `server` to `true`. That matches the Nuxt module default for `useConvexQuery`.

## Use the composables

```vue [src/components/TaskList.vue]
<script setup lang="ts">
import { useConvexQuery } from '@onmax/convex-vue'
import { api } from '../convex/_generated/api'

const { data, suspense } = useConvexQuery(api.tasks.list, { userId: 'demo' })
await suspense()
</script>
```
