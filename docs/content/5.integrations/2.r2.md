# R2 Integration

Use the Convex R2 component with Nuxt Convex for uploads and metadata sync.

## 1. Install Dependencies

```bash
pnpm add @convex-dev/r2
```

> Requires `@convex-dev/r2` **>= 0.9.1** for the Vue upload hook.

## 2. Enable in Nuxt

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['nuxt-convex'],
  convex: {
    r2: true,
  },
})
```

This enables the `#convex/r2` virtual module and auto-imports `useConvexR2Upload`.
Nuxt Convex also bridges the Convex Vue context so `useUploadFile` works without extra setup.

## 3. Install the Convex Component

Create `convex/convex.config.ts` and install the R2 component:

```ts [convex/convex.config.ts]
import r2 from '@convex-dev/r2/convex.config.js'
import { defineApp } from 'convex/server'

const app = defineApp()
app.use(r2)

export default app
```

Set your R2 credentials in the Convex environment (see the R2 component README for details).

## 4. Create R2 Client API

Create an API export for the client hook:

```ts [convex/r2.ts]
import type { DataModel } from './_generated/dataModel'
import { R2 } from '@convex-dev/r2'
import { components } from './_generated/api'

export const r2 = new R2(components.r2)

export const { generateUploadUrl, syncMetadata } = r2.clientApi<DataModel>({
  checkUpload: async (_ctx, _bucket) => {
    // validate permissions
  },
  onUpload: async (_ctx, _bucket, _key) => {
    // handle post-upload
  },
})
```

## 5. Upload From Nuxt

### Using the R2 Vue Hook

```vue
<script setup lang="ts">
import { api } from '#convex/api'
import { useUploadFile } from '#convex/r2'

const uploadFile = useUploadFile(api.r2)

async function onFileSelected(file: File) {
  const key = await uploadFile(file)
  console.log('Uploaded key:', key)
}
</script>
```

### Using the Nuxt Wrapper

```vue
<script setup lang="ts">
import { api } from '#convex/api'

const { upload, isUploading, progress, error } = useConvexR2Upload(api.r2, {
  onSuccess: key => console.log('Uploaded key:', key),
})

async function onFileSelected(file: File) {
  await upload(file)
}
</script>
```
