title: useConvexConnectionState
description: Monitor the WebSocket connection between the Convex client and backend.
---

`useConvexConnectionState` subscribes to the connection state of the Convex realtime client.

```vue [src/components/ConnectionBanner.vue]
<script setup lang="ts">
import { useConvexConnectionState } from '@onmax/convex-vue'

const { state, isConnected } = useConvexConnectionState()
</script>
```

## Signature

```ts
function useConvexConnectionState(): {
  state: Readonly<Ref<ConnectionState | null>>
  isConnected: ComputedRef<boolean>
}
```

## Important behavior

- Returns `null` when no realtime client is connected.
- `isConnected` is a convenience shorthand for `state.value?.isWebSocketConnected`.
- Automatically re-subscribes when the client changes (e.g. after `controller.reconfigure()`).
