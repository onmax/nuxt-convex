---
title: useConvexClient
description: Access the active realtime client from the advanced Vue entrypoint.
---

`useConvexClient` lives in `@onmax/convex-vue/advanced`. Use it when the higher-level composables are not enough and you need the raw realtime client directly.

```ts [app/composables/convex.ts]
import { useConvexClient } from '@onmax/convex-vue/advanced'

const client = useConvexClient()
```

## Signature

```ts
function useConvexClient(): ConvexClient
```

## Important behavior

- The composable throws when no realtime client is connected.
- [`useConvexController`](/api-reference/use-convex-controller) handles delayed-connection flows before you call this composable.
- The higher-level composables still cover most application code. Use this escape hatch only when you need direct client access.
