title: useConvexController
description: Access the advanced controller entrypoint for delayed connection, reconfiguration, and explicit raw client helpers.
---

`useConvexController` returns the advanced controller from `@onmax/convex-vue/advanced`. The controller manages delayed connection, runtime reconfiguration, and explicit disconnect flows.

```vue [app/components/TaskInspector.vue]
<script setup lang="ts">
import { useConvexController } from '@onmax/convex-vue/advanced'
import { onMounted } from 'vue'

const controller = useConvexController()

onMounted(() => {
  controller.connect({
    url: window.localStorage.getItem('convex-url') || 'https://your-app.convex.cloud',
  })
})
</script>
```

## Behavior

- `connect()` requires a URL and creates the HTTP client plus the browser realtime client when running in the browser.
- `reconfigure()` updates the active connection without exposing internal refs.
- `disconnect()` clears the active connection.
- Use [`useConvexClient`](/api-reference/use-convex-client) and [`useConvexHttpClient`](/api-reference/use-convex-http-client) when you need the raw clients.

## Use it for

- delayed connection after app startup
- host-provided or multi-tenant runtime configuration
- pairing with the advanced raw-client composables when the higher-level API is not enough
