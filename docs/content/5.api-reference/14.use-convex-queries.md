---
title: useConvexQueries
description: Subscribe to multiple dynamic queries as a reactive record.
---

`useConvexQueries` manages multiple query subscriptions from a single reactive object. Subscriptions are added, removed, and updated automatically as the input changes.

```vue [src/components/Dashboard.vue]
<script setup lang="ts">
import { useConvexQueries } from '@onmax/convex-vue'
import { api } from '../convex/_generated/api'

const { data, errors } = useConvexQueries(computed(() => ({
  tasks: { query: api.tasks.list, args: { userId: 'user_123' } },
  stats: { query: api.stats.get, args: {} },
})))
</script>
```

## Signature

```ts
function useConvexQueries<T extends Record<string, QueryEntry>>(
  queries: MaybeRefOrGetter<T>,
): {
  data: Readonly<Ref<{ [K in keyof T]?: FunctionReturnType<...> }>>
  errors: Readonly<Ref<Record<string, Error | null>>>
}
```

## Important behavior

- Each entry supports `"skip"` to conditionally disable individual queries.
- Only entries whose args change are re-subscribed.
- Adding or removing keys from the record subscribes or unsubscribes automatically.
- All subscriptions are cleaned up on scope disposal.
