---
title: useConvexClient
description: Access the raw Convex browser client when the higher-level composables are not enough.
---

`useConvexClient` returns the realtime `ConvexClient` instance from the shared Vue context.

```vue [app/components/TaskInspector.vue]
<script setup lang="ts">
import { api } from '#convex/api'

const client = useConvexClient()

const task = await client.query(api.tasks.get, {
  id: 'task_id',
})
</script>
```

## Behavior

- throws if the browser client has not been initialized
- only returns the realtime client, not the SSR HTTP client
- works in Nuxt and Vue once the shared context exists

## Use it for

- one-off calls outside the composable wrappers
- custom subscription flows
- direct access to lower-level Convex client methods
