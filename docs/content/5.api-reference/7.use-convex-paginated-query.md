---
title: useConvexPaginatedQuery
description: Subscribe to paginated Convex queries on the client and load more pages incrementally.
---

`useConvexPaginatedQuery` is the client-only helper for Convex functions that return a `PaginationResult`.

```vue [app/components/TaskFeed.vue]
<script setup lang="ts">
import { api } from '#convex/api'

const {
  data,
  pages,
  isDone,
  isPending,
  isLoadingMore,
  loadMore,
  reset,
  suspense,
} = useConvexPaginatedQuery(api.tasks.listPaginated, {}, { numItems: 20 })

await suspense()
</script>
```

## Signature

```ts
function useConvexPaginatedQuery<Query extends FunctionReference<'query'>>(
  query: Query,
  args: MaybeRefOrGetter<Omit<FunctionArgs<Query>, 'paginationOpts'>>,
  options: {
    numItems: number
  },
): {
  suspense: () => Promise<unknown[][]>
  pages: ComputedRef<unknown[][]>
  data: ComputedRef<unknown[]>
  lastPage: ComputedRef<PaginationResult<unknown> | undefined>
  error: Ref<Error | null>
  isDone: Ref<boolean>
  isPending: ComputedRef<boolean>
  isLoadingMore: Ref<boolean>
  loadMore: () => void
  reset: () => void
}
```

## Important behavior

- This helper does not run the SSR prefetch path.
- `loadMore()` subscribes to the next page.
- `reset()` clears pages and reloads the first page.
- Certain pagination errors trigger an automatic internal reset.
