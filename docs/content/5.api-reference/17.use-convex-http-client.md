---
title: useConvexHttpClient
description: Access the active HTTP client from the advanced Vue entrypoint.
---

`useConvexHttpClient` lives in `@onmax/convex-vue/advanced`. Use it when you need direct access to the Convex HTTP client outside the higher-level composables.

```ts [app/composables/convex.ts]
import { useConvexHttpClient } from '@onmax/convex-vue/advanced'

const client = useConvexHttpClient()
```

## Signature

```ts
function useConvexHttpClient(): ConvexHttpClient
```

## Important behavior

- The composable throws when no HTTP client is connected.
- [`useConvexController`](/api-reference/use-convex-controller) handles the same delayed-connection flows that this composable depends on.
- `useConvexQuery().suspense()` remains the preferred API unless you need direct transport access.
