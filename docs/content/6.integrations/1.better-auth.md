title: Better Auth

---

`nuxt-convex` registers a `convex` Better Auth database provider through the `better-auth:database:providers` hook. `@onmax/nuxt-better-auth` can then generate its database adapter from that provider.

::important
This integration runs on the Nuxt server. It is not available in a client-only setup.
::

## Current support status

This integration is currently dependency-constrained. The provider hook and HTTP adapter ship in this repo, but the end-to-end stack still depends on external packages that the workspace tracks through its current catalog pins.

Treat this page as documentation for the shipped wiring, not as a stable release matrix. If you need a known-good combination today, match the package versions used by the playground in this repository.

The page stays in the main docs because the adapter and provider wiring already ship in this repository.

## Install the Better Auth stack

```bash
pnpm add @onmax/nuxt-better-auth @convex-dev/better-auth better-auth
```

Use versions that match the compatibility window you are targeting. The Nuxt-side provider in this repo expects a Better Auth stack that can consume the generated `convex` database provider.

## Configure Nuxt

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['nuxt-convex', '@onmax/nuxt-better-auth'],
  auth: {
    database: {
      provider: 'convex',
    },
  },
})
```

`nuxt-convex` writes the Convex URL into runtime config and also supports `auth.database.convexUrl` as a higher-priority override.

## Add environment variables

```bash [.env]
CONVEX_URL=https://your-project.convex.cloud
BETTER_AUTH_SECRET=replace-with-a-secret-at-least-32-characters
```

## Create auth config files

```ts [server/auth.config.ts]
import { defineServerAuth } from '@onmax/nuxt-better-auth/config'

export default defineServerAuth(() => ({
  appName: 'nuxt-convex app',
}))
```

```ts [app/auth.config.ts]
import { defineClientAuth } from '@onmax/nuxt-better-auth/config'

export default defineClientAuth({})
```

## What ships in this repo

`nuxt-convex` exports `createConvexHttpAdapter` from `nuxt-convex/better-auth`. The adapter validates the Convex URL, maps Better Auth record shapes, and paginates reads through the Convex HTTP client.
