---
title: R2
description: Connect Cloudflare R2 to Convex and consume the upload helpers from Nuxt through the R2 integration path.
---

Use the Convex R2 component when you want bucket-backed uploads instead of Convex storage objects.

## Install the dependencies

```bash
pnpm add @convex-dev/r2
```

## Enable Nuxt support

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['nuxt-convex'],
  convex: {
    r2: true,
  },
})
```

## Register the Convex component

```ts [convex/convex.config.ts]
import r2 from '@convex-dev/r2/convex.config.js'
import { defineApp } from 'convex/server'

const app = defineApp()
app.use(r2)

export default app
```

## Create the client API

```ts [convex/r2.ts]
import type { DataModel } from './_generated/dataModel'
import { R2 } from '@convex-dev/r2'
import { components } from './_generated/api'

export const r2 = new R2(components.r2)

export const { generateUploadUrl, syncMetadata } = r2.clientApi<DataModel>({
  checkUpload: async (_ctx, _bucket) => {},
  onUpload: async (_ctx, _bucket, _key) => {},
})
```

## Use the Nuxt helpers

```vue [app/components/R2Upload.vue]
<script setup lang="ts">
import { api } from '#convex/api'
import { useUploadFile } from '#convex/r2'

const directUpload = useUploadFile(api.r2)
const { upload, progress } = useConvexR2Upload(api.r2)
</script>
```

`#convex/r2` exposes the upstream R2 Vue helpers. `useConvexR2Upload` adds the Nuxt-friendly upload state wrapper documented in this repo.
