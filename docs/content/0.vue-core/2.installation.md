---
title: Vue Installation
description: Install the standalone Vue package.
---

Install the standalone Vue package with Convex and Vue:

```bash
pnpm add @onmax/convex-vue convex vue
```

Register the plugin in your Vue app:

```ts
import { convexVue } from '@onmax/convex-vue'
import { createApp } from 'vue'
import App from './App.vue'

const app = createApp(App)

app.use(convexVue, {
  url: import.meta.env.VITE_CONVEX_URL!,
})

app.mount('#app')
```

If you are using Nuxt, install `nuxt-convex` instead and let the module wire the plugin automatically.
