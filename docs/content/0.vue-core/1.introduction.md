---
title: Vue Core
description: Standalone Vue composables for Convex.
---

`@onmax/convex-vue` is the standalone Vue layer extracted from `nuxt-convex`.

It provides the canonical plugin and composables:

- `convexVue`
- `useConvexClient`
- `useConvexQuery`
- `useConvexMutation`
- `useConvexAction`
- `useConvexPaginatedQuery`
- `useConvexStorage`
- `useConvexUpload`

Use it directly in plain Vue apps, or consume it indirectly through the Nuxt module.
