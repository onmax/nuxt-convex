---
title: Installation
description: Install nuxt-convex or @onmax/convex-vue, connect Convex, and generate the typed API used throughout the docs.
---

Install the package that matches your app runtime, then start Convex so the generated API exists before you write frontend code.

## Nuxt track

Use `nuxt-convex` when your app runs on Nuxt and you want the module to wire the shared Vue package for you.

::code-group{sync="pm"}

```bash [pnpm]
pnpm add nuxt-convex convex
```

```bash [npm]
npm install nuxt-convex convex
```

```bash [yarn]
yarn add nuxt-convex convex
```

::

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['nuxt-convex'],
})
```

The module reads `CONVEX_URL` first, then falls back to `NUXT_PUBLIC_CONVEX_URL`.

## Vue track

Use `@onmax/convex-vue` when you want to install the plugin directly in Vue 3.

::code-group{sync="pm"}

```bash [pnpm]
pnpm add @onmax/convex-vue convex
```

```bash [npm]
npm install @onmax/convex-vue convex
```

```bash [yarn]
yarn add @onmax/convex-vue convex
```

::

```ts [src/main.ts]
import { convexVue } from '@onmax/convex-vue'
import { createApp } from 'vue'
import App from './App.vue'

const app = createApp(App)

app.use(convexVue, {
  url: import.meta.env.VITE_CONVEX_URL,
})

app.mount('#app')
```

## Generate the API

Run the Convex development server after installing either track.

```bash
npx convex dev
```

That command creates the generated files used by the examples in this documentation:

- `convex/_generated/server`
- `convex/_generated/api`
- `#convex/api` in Nuxt apps

::important
If `#convex/api` is missing in a Nuxt app, `nuxt-convex` will warn during startup. The warning is accurate. Run `npx convex dev` in the project that owns your Convex directory.
::

## Verify the setup

Use the Nuxt version when you installed `nuxt-convex`.

```vue [app/pages/index.vue]
<script setup lang="ts">
import { api } from '#convex/api'

const { data, suspense } = useConvexQuery(api.tasks.list, { userId: 'demo' })
await suspense()
</script>
```

Use the Vue version when you installed `@onmax/convex-vue`.

```vue [src/App.vue]
<script setup lang="ts">
import { useConvexQuery } from '@onmax/convex-vue'
import { api } from '../convex/_generated/api'

const { data, suspense } = useConvexQuery(api.tasks.list, { userId: 'demo' })
await suspense()
</script>
```
