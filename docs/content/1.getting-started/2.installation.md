Add `nuxt-convex` to your Nuxt project with the required Convex peer dependency.

## Setup

Install the module and Convex:

::code-group{sync="pm"}

```bash [pnpm]
pnpm add nuxt-convex convex
```

```bash [npm]
npm install nuxt-convex convex
```

```bash [yarn]
yarn add nuxt-convex convex
```

::

::note
The standalone Vue integration lives in `@onmax/convex-vue`. The Nuxt module installs and wires it for you.
::

Add the module to your Nuxt configuration:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['nuxt-convex'],
  convex: {
    storage: true, // Enable file storage (optional)
  },
})
```

## Environment Variables

The module reads `CONVEX_URL` or `NUXT_PUBLIC_CONVEX_URL` automatically.

::note
Running `npx convex dev` starts the Convex development server and sets `CONVEX_URL` automatically.
::

| Variable                 | Description                                     |
| ------------------------ | ----------------------------------------------- |
| `CONVEX_URL`             | Convex deployment URL (set by `npx convex dev`) |
| `NUXT_PUBLIC_CONVEX_URL` | Alternative, follows Nuxt convention            |

## Virtual Modules

The module provides two virtual imports:

| Import        | Purpose                     |
| ------------- | --------------------------- |
| `#convex`     | Core composables and client |
| `#convex/api` | Generated Convex API types  |

```ts
import { useConvexQuery } from '#convex'
import { api } from '#convex/api'
```
