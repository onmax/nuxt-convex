---
title: Connection State
description: Monitor the WebSocket connection between the client and the Convex backend.
---

`useConvexConnectionState` subscribes to the connection state of the Convex realtime client. Use it to show connectivity indicators or pause actions while offline.

## Basic usage

```vue [src/components/ConnectionBanner.vue]
<script setup lang="ts">
import { useConvexConnectionState } from '@onmax/convex-vue'

const { state, isConnected } = useConvexConnectionState()
</script>

<template>
  <div v-if="!isConnected" class="banner">
    Reconnecting...
  </div>
</template>
```

## Return values

::field-group
::field{name="state" type="Readonly<Ref<ConnectionState | null>>"}
The full connection state from the Convex SDK. `null` when no client is connected.
::

::field{name="isConnected" type="ComputedRef<boolean>"}
`true` when the WebSocket is connected.
::
::

The `ConnectionState` object includes:

- `isWebSocketConnected` — Whether the WebSocket is open.
- `hasInflightRequests` — Whether there are pending requests.
- `hasEverConnected` — Whether the client has connected at least once.
- `connectionCount` — Total number of connections.
- `inflightMutations` — Number of mutations in flight.
