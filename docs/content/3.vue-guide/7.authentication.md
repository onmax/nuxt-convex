title: Authentication
description: Track auth state and conditionally render UI based on whether the user is authenticated.

`useConvexAuth` tracks the authentication state of the Convex client. Use it to conditionally render UI or skip queries until the user is signed in.

## Set up authentication

Call `useConvexAuth` with a `fetchToken` option at the root of your app. The composable configures the Convex client to use the provided token fetcher.

```vue [src/App.vue]
<script setup lang="ts">
import { useConvexAuth } from '@onmax/convex-vue'

const { isLoading, isAuthenticated } = useConvexAuth({
  fetchToken: async ({ forceRefreshToken }) => {
    // Return a JWT from your auth provider, or null if not signed in
    return await myAuthProvider.getToken({ forceRefreshToken })
  },
})
</script>
```

## Read auth state

Child components call `useConvexAuth()` without options to read the shared state.

```vue [src/components/Nav.vue]
<script setup lang="ts">
import { useConvexAuth } from '@onmax/convex-vue'

const { isLoading, isAuthenticated } = useConvexAuth()
</script>

<template>
  <nav>
    <span v-if="isLoading">Loading...</span>
    <span v-else-if="isAuthenticated">Welcome back!</span>
    <span v-else>Please sign in</span>
  </nav>
</template>
```

## Renderless components

The library provides three renderless components for template-driven conditional rendering.

```vue [src/App.vue]
<template>
  <ConvexAuthLoading>
    <p>Checking authentication...</p>
  </ConvexAuthLoading>

  <ConvexAuthenticated>
    <Dashboard />
  </ConvexAuthenticated>

  <ConvexUnauthenticated>
    <LoginPage />
  </ConvexUnauthenticated>
</template>
```

- `<ConvexAuthenticated>` — renders its slot when the user is authenticated.
- `<ConvexUnauthenticated>` — renders its slot when the user is not authenticated and auth is not loading.
- `<ConvexAuthLoading>` — renders its slot while auth state is being determined.

## Skip queries until authenticated

Combine `useConvexAuth` with the `"skip"` pattern to avoid running queries before the user is signed in.

```vue [src/components/Dashboard.vue]
<script setup lang="ts">
import { useConvexAuth, useConvexQuery } from '@onmax/convex-vue'
import { api } from '../convex/_generated/api'

const { isAuthenticated } = useConvexAuth()

const { data } = useConvexQuery(
  api.tasks.myTasks,
  computed(() => isAuthenticated.value ? {} : 'skip'),
)
</script>
```
